# CONNECT FOUR

### Description
A minimalist connect four game

###Tech used / Stack
* HTML
* CSS
* Javascript
* jQuery
* github.com/daneden/animate.css

###Features
* Interactive board
* Players name input
* Game ends when there is a winner or tie
* Wins tracker
* Can reset game to initial state
* Enter key works after typing in players name

###Future Implementations
* A.I.
